# UI Documentation

Use this folder for UI implementation planning and design decisions.

- `STITCH_TO_ANDROID_MAPPING.md`: traceability between Stitch exports and XML/fragments.
- `STITCH_UI_ADAPTATION.md`: concrete mapping from `ui_exports/stitch_UI` exports into app layouts.
- future screen-state docs (loading/error/empty/offline per screen).
